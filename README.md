# Caramelo Tech .github

Este é o repositório de configuração da organização **Caramelo Tech** no GitHub.

## O que tem aqui?

- `profile/README.md` - O README exibido na [página principal da organização](https://github.com/caramelotech)
- `profile/en-US.md` - Versão em inglês do README de perfil
- `ISSUE_TEMPLATE/` - Templates para issues (bug report e feature request)
- `PULL_REQUEST_TEMPLATE.md` - Template para pull requests
- `CODE_OF_CONDUCT.md` - Código de conduta da comunidade
- `CONTRIBUTING.md` - Guia de contribuição

## Sobre o repositório .github

O repositório `.github` é um repositório especial do GitHub que permite configurar arquivos padrão para toda a organização, como:

- README de perfil da organização
- Templates de issues e pull requests
- Arquivos de saúde da comunidade (CODE_OF_CONDUCT, CONTRIBUTING, etc.)
- Workflows reutilizáveis do GitHub Actions
